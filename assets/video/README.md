# Videos & Animationen

## Benötigte Videos

### 1. Hero Video
- **Dateiname:** `hero.mp4` (und `hero.webm` als Fallback)
- **Spezifikationen:**
  - Auflösung: 1920x1080px (Full HD)
  - Dauer: 10-20 Sekunden (Loop)
  - Format: MP4 (H.264), WebM
  - Dateigröße: < 5MB
  - Autoplay, Muted, Loop
- **Inhalt:** Atmosphärische Aufnahme von Lebkuchenhäusern, Weihnachtsmarkt oder Dekoration

### 2. Erklärvideo
- **Dateiname:** `explanation.mp4`
- **Spezifikationen:**
  - Auflösung: 1280x720px (720p)
  - Dauer: 30-60 Sekunden
  - Format: MP4 (H.264)
  - Dateigröße: < 10MB
  - Mit Audio (optional)
- **Inhalt:** Kurze Anleitung, wie man ein Lebkuchenhaus baut/dekoriert

## Optimierung

Verwende Tools wie:
- **HandBrake** für Video-Kompression
- **FFmpeg** für Format-Konvertierung
- **Online Tools:** CloudConvert, etc.

## Alternative: Intro Animation

Falls kein Intro-Video verfügbar ist, kann die CSS/SVG-Animation verwendet werden (bereits im Code implementiert).
