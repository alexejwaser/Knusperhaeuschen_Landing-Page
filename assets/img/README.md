## Bildspezifikationen

### Konfigurator-Bilder (PNG mit Transparenz)
- **Größe:** 400x400px
- **Format:** PNG mit Alpha-Kanal
- **Optimierung:** TinyPNG oder ähnlich

### Team-Fotos
- **Größe:** 300x300px (mindestens)
- **Format:** JPG oder WebP
- **Aspect Ratio:** 1:1 (quadratisch)

## Aus Vorgaben-Ordner verwenden

Nutze die Assets aus `../Knusperhaeuschen Vorgaben/`:
- Keyvisuals → house-base.png
- Papiertexturen → Hintergründe für Layers
- Logo → Navigation (optional)
